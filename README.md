# ScratchLearn – версія React + Node (Express)

Це навчальний прототип інтерактивної платформи Scratch для учнів середньої школи з елементами гейміфікації:
- Теорія (уроки)
- Практика (завдання різних рівнів)
- Кабінет (бали/рівень/бейджі/прогрес)
- Рейтинг
- Модуль вчителя (створення завдань)

## Вимоги
- Node.js 18+ (рекомендовано 20+)
- npm

## Запуск (2 термінали)
### 1) Сервер
```bash
cd server
npm i
npm run dev
```
Сервер підніметься на http://localhost:5179

### 2) Клієнт (React)
```bash
cd client
npm i
npm run dev
```
Клієнт: http://localhost:5173

> Клієнт підключається до API за замовчуванням: http://localhost:5179/api

## Дані
Сервер зберігає дані у файлі:
- `server/data.json`

## Ролі
- Учень (student) – виконує завдання, отримує бали, потрапляє в рейтинг
- Вчитель (teacher) – має вкладку «Вчитель», може створювати завдання

## Як працює зарахування завдання (прототип)
Scratch-редактор не віддає блоки напряму, тому прототип використовує:
- чек-лист (має бути відмічений весь),
- мініквіз (1 питання – правильна відповідь).

## Швидкий тест
1) Зареєструй учня (role: student) і виконай 1 завдання
2) Зареєструй другого учня – рейтинг почне наповнюватися
3) Зареєструй вчителя – зможеш створити нові завдання
